# Easy git commands

* basic
```Shell
git init
```
```Shell
git add .
git add -A
```
```Shell
git status
git log
```
```Shell
git commit -m "~~~"
```
```Shell
git remote add origin https:/github-site
git remote
git push -u origin main //set default
git push
```
```Shell
//cancel add
git reset -- <filename>
```
```Shell
//discard change (reset first)
git checkout -- <filename>
```
```Shell
//undo last commit
git reset -- HEAD~1
```
